Graph BFS

Breadth-First Search traversal with queue visualization

// THEORY

BFS (Breadth-First Search) explores a graph level by level using a queue. Starts at a source node and visits all neighbors before going deeper.

  • Time: O(V + E) — vertices + edges
  • Space: O(V) — queue storage
  • Uses: shortest path, level-order, web crawling, social networks
  • Uses Queue (FIFO) internally
