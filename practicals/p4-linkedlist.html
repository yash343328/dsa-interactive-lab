Linked List

Singly Linked List — Insert Head/Tail · Delete · Traverse

// THEORY

A Linked List is a linear structure where each node contains data and a pointer to the next node. No contiguous memory required.

  • Insert at head/tail: O(1)
  • Insert at position: O(n)
  • Delete: O(n) — need to find the node
  • Search: O(n)
  • Memory efficient — dynamic size
