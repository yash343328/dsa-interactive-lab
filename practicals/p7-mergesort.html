Merge Sort

Divide & Conquer — animated splitting and merging phases

// THEORY

Merge Sort divides the array into halves, recursively sorts each half, then merges them back in sorted order.

  • Time: O(n log n) — all cases
  • Space: O(n) — extra array needed
  • Stable, reliable, great for linked lists
  • Best for large datasets where stability matters
