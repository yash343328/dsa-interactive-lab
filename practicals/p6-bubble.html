Bubble Sort

Step-by-step animated sorting with comparison & swap counters

// THEORY

Bubble Sort repeatedly compares adjacent elements and swaps them if they are in the wrong order. Largest elements "bubble up" to end.

  • Time: O(n²) average & worst, O(n) best (optimized)
  • Space: O(1) — in-place
  • Stable sort — equal elements maintain relative order
Speed: 500ms
