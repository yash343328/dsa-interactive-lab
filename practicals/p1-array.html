Array Operations

Insert · Delete · Search · Traverse — with live visualization

// THEORY

An Array is a contiguous block of memory storing elements of the same type, accessible via index.

  • Access: O(1) — direct index
  • Search: O(n) — linear scan
  • Insertion/Deletion: O(n) — shifting required
