Binary Search Tree

Insert · Search · Inorder/Preorder/Postorder Traversal

// THEORY

A BST has the property: left child < parent < right child. Enables fast search.

  • Insert/Search: O(log n) average, O(n) worst
  • Inorder traversal gives sorted output
  • Used in: databases, file systems, priority queues
