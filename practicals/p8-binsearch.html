Binary Search

Efficient search on sorted arrays — Low · Mid · High pointer animation

// THEORY

Binary Search works on sorted arrays by repeatedly halving the search space.

  • Time: O(log n) — much faster than linear O(n)
  • Space: O(1) iterative, O(log n) recursive
  • Prerequisite: array MUST be sorted
  • Example: searching in 1000 elements takes max 10 steps!
