Queue

First In First Out (FIFO) — Enqueue · Dequeue · Front · Rear

// THEORY

A Queue is a FIFO (First In, First Out) data structure. Elements are added at the REAR and removed from the FRONT.

  • Enqueue: add to rear — O(1)
  • Dequeue: remove from front — O(1)
  • Use cases: CPU scheduling, BFS, print queues, task management
